# Gym Management System

## Overview

Gym Management System is a web application built with ASP.NET Core MVC that helps manage a fitness center efficiently. 
The system provides features for managing members, trainers, subscriptions, and attendance through a clean and user-friendly interface.

---

## Features

* Member Management (Create, Edit, Delete, View)
* Trainer Management (Create, Edit, Delete, View)
* Subscription Plans
* Attendance Tracking
* User Authentication & Authorization
* Responsive User Interface
* CRUD Operations
* Form Validation
* SQL Server Database Integration

---

## Technologies Used

* ASP.NET Core MVC
* C#
* Entity Framework Core
* SQL Server
* HTML5
* CSS3
* Bootstrap
* JavaScript
* LINQ
* Repository Pattern
* N-Tier Architecture

---

## Project Structure

* **GymManagment.PL** – Presentation Layer
* **GymManagment.BLL** – Business Logic Layer
* **GymManagment.DAL** – Data Access Layer
